Formato Recomendado para Subida de Usuarios

El archivo Excel debe contener únicamente las siguientes columnas y respetar el formato indicado para el Rut.

Importante: El Rut debe tener el formato 11111111-1 (sin puntos, con guion).
N° Rut * Nombre * Año Ingreso Sist. Ingreso E-mail * Fono
1 11111111-1 NOMBRE_ALUMNO 1 2018 SELECCIONADO alumno1@correo.cl 912345678
2 22222222-2 NOMBRE_ALUMNO 2 2020 CONVENIO alumno2@correo.cl 923456789
3 33333333-3 NOMBRE_ALUMNO 3 2023 BECA alumno3@correo.cl 934567890
  • Solo se consideran las columnas resaltadas: Rut, Nombre y E-mail.
  • El resto de las columnas serán ignoradas.
  • El archivo debe estar en formato .xlsx o .xls.
  • El Rut debe tener el formato 11111111-1 (sin puntos, con guion).
